# GRANJA DE IAs — multi_farm_system

## Arquitectura
Sistema evolutivo competitivo con múltiples granjas de agentes que generan ingresos reales.
Documento de referencia: GRANJA_DE_IAs_v2 (arquitectura v2.0)

## Stack
- Python 3.11+
- SQLite (observatory/memory)
- Claude API claude-sonnet-4-20250514 (mutación de estrategias)
- Rich (dashboard terminal)

## Estructura de carpetas objetivo
```

  main.py
  config.py
  /core

    competition.py

  /farms

```

## Comandos
- `python main.py` — arranca el loop global
- `python -m pytest tests/` — corre tests

## Convenciones
- Dataclasses para todos los modelos (models.py)
- Estrategias siempre como dict JSON-serializable
- Circuit breaker OBLIGATORIO antes de cualquier llamada API externa
- Logging estructurado en cada acción económica
- Sin valores hardcodeados — todo en config.py
- Variables de entorno para API keys, nunca en código
